TabCopy now has a Fancy Mode that lets you select a format when you're copying tabs.
Try it out by selecting and customizing it below.
You can always switch back to the familiar Simple Mode if Fancy Mode is not your cup of tea.

Fancy Mode
Single clicking copies the selected tab•Double clicking copies all tabs in the window
Triple clicking copies all tabs in all windows
Clicking opens a popup that lets you select the range of tabs to copy
and the format to use

Copy confirmation
Ignore pinned tabs
Show in context menu
Shortcut key
hit once to copy selected tab
hit twice to copy all tabs in the window
hit three times to copy all tabs in all windows
